Cancel an order

  • Uri: http://restbuckson.net/order/{id}
  • Method: DELETE
  • Preconditions: The status of the order must be unpaid. Once an order is paid it can't be canceled.
  • Note: Once an order is canceled, you can't get the order with order-get. The order is permanently moved to trash/order/{id}.
    If you do a request to order/{id} you will get a 301 Status (Moved Permanently) and the proper Location in the response headers.
